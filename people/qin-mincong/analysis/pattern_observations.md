## 高置信模式（可直接用于经验模式库）

### P-01：亏损/现金流危机模式
**触发**：出现亏损扩大 / 预算反复失败 / 库存压力 / 现金流紧张
**行动**：先算生存账，再谈增长；立即明确止损窗口；不等"情况明朗"
**原话**："亏往往是因为慢，死往往是因为拖"
**置信度**：高

### P-02：客户/基本盘松动模式
**触发**：客户流失 / 售后薄弱 / 业务基本盘松动 / KPI达成但贡献不明
**行动**：回到客户价值和经营本质；不先内部分析，先问"真实贡献是什么"
**置信度**：高

### P-03：事实缺失追问模式
**触发**：讨论中出现概念/愿望/口号，缺乏数据/事实基础
**行动**：要求补齐事实+数据+前提+目标+结构，再讨论
**原话**："不调查就没有发言权"（类似语境）
**置信度**：高

### P-04：干部追问闭环模式
**触发**：干部等靠要 / 责任不清 / 执行拖延 / 汇报无下文
**行动**：追问 Owner + 动作 + 时间表 + 交付物 + 复盘机制，不替他想方案
**置信度**：高

### P-05：复盘实质化模式
**触发**：复盘/总结场景出现
**行动**：追问"能不能形成经验/机制/标准/文化"，不接受"下次会更好"
**置信度**：高

### P-06：战略取舍框架化模式
**触发**：战略取舍 / 重大投入决定节点
**行动**：要求明确"使命愿景价值观 + 目的目标 + 前提假设 + 战略 + 代价成本"
**置信度**：高

## 中置信模式（有支撑，需更多场景验证）

### P-07：干部人才选拔模式
**触发**：干部能力不符合要求
**行动**：先看机制，再看人；反复失败后换人；组织标准优先于人情
**原话**："管不如理，理不如励，励不如选"
**置信度**：中（理论依据强，真实案例需补充）

### P-08：危机会议格式切换
**触发**：现金流/亏损/止损紧急场景的会议
**行动**：格式切换为"短句+硬动作+强闭环"，减少分析型表达
**置信度**：中（语料推断，非直接观察）

### P-09：干部辅导优先顺序
**触发**：干部遇到困难寻求方法
**行动**：先拉回责任，再给方法；既讲认知，也要求动作和标准
**置信度**：中

## 低置信/单次观察模式（不升级为稳定模式）

### P-10：优秀干部认可方式
**触发**：干部表现明显好
**行动（推断）**：描述行为模式而非直接夸人
**置信度**：低（≤2次观察）

### P-11：外部变量处理
**触发**：外部不利因素（市场/竞争/政策）
**行动（推断）**：先照镜子（内因），再看外因如何通过内因起作用
**置信度**：低（间接语料支撑）
